# System 1.0 · 系统定位与运行宪法（定稿版）

## 一、系统一句话定义（冻结）

> **System 1.0 是一个 AI 原生的个人原型推进系统。  
> 它与我共同思考“做什么、值不值得做”，  
> 并在决策确认后，自动、可控地把想法推进到可验证原型。**

- **第一服务对象：我自己**
- **目标状态：我每天都在用**
- **对外扩展：自然外溢，而非强设计**

---

## 二、系统核心价值（不再争论）

System 1.0 解决的不是“写代码”，而是以下两个问题：

1. **当我有一个想法时，不知道该做什么**
2. **当我做了决定，却很难把它真正跑到一个可验证状态**

系统的价值不在于“做得多复杂”，而在于：

> **让我在一周内，把一个模糊想法变成一个“能看、能试、能判断去留”的原型。**

---

## 三、人机分工原则（最重要的边界）

### 1️⃣ 思考阶段（允许聪明与不确定）

- 人与系统共同参与
- 系统可以：
  - 提出不同观点
  - 挑战假设
  - 暴露风险
- **最终决策权始终在人**

这是探索，不追求稳定。

---

### 2️⃣ 执行阶段（允许笨，但必须可控）

- 决策一旦确认
- 执行完全交给系统
- 系统必须：
  - 遵守已冻结规则
  - 不擅自改变方向
  - 不重新讨论已决策事项

这是落地，追求**确定性与可复现性**。

---

## 四、阶段分层与职责（承认并冻结）

### 阶段 1：系统骨架（已完成，冻结）
- 多智能体结构
- Skill 体系
- Failure Routing
- Dispatch 作为唯一交接物

**❌ 不允许在 System 1.0 中修改**

---

### 阶段 2：世界规则（已完成，冻结）
阶段 2 的成果不是“文档”，而是：

- 技术栈选择 → **可执行规则**
- 产品形态 → **硬约束**
- 接口 / 冒烟 / 部署 → **系统前提**

阶段 2 决定的是：

> **在这个系统世界里，什么是允许发生的，什么不是。**

**❌ System 1.0 中不再重新讨论这些问题**

---

### 阶段 3：执行引擎（System 1.0 的主体）

阶段 3 的唯一职责是：

> **在阶段 2 已冻结的规则空间内，  
> 自动生成、执行、迭代“原型实例”。**

关键词不是“决策”，而是：

- 实例化
- 参数化
- 自动推进

---

## 五、System 1.0 的对外体验（Execution Round）

对“使用者（我自己）”而言，系统只暴露一个概念：

### **Execution Round（执行轮次）**

每一轮包含：

1. 输入
   - 自然语言想法  
   - 或基于已有结果的调整说明
2. 系统自动执行（中间过程默认不可见）
3. 输出
   - 一个**可直观看到的原型结果**
   - 明确的版本标识

我只需要关心：

> **这一轮跑出来的东西，我要不要继续？**

---

## 六、调整与迭代原则（非常关键）

### 允许的调整
- 原型范围（Prototype Scope）
- 执行参数（Execution Parameters）
- 表现形式、路径、交互

### 禁止的调整（System Bug）
- 推翻产品形态
- 更换技术栈
- 绕过 Playbook
- 绕过 Failure Routing

> **System 1.0 的“自由度”只存在于实例层，不存在于规则层。**

---

## 七、系统如何“越用越好用”（明确限定）

System 1.0 的自我优化只发生在：

- 更好的默认参数
- 更高成功率的执行路径
- 更稳定的外部工具（MCP）选择
- 更清晰的失败归因与复盘

**系统不会：**
- 偷偷改变规则
- 擅自替我做方向决策
- 用“成功率”压制探索

---

## 八、System 1.0 的非目标（明确声明）

System 1.0 **不是**：

- 一个通用低代码平台
- 一个自动创业系统
- 一个替人做决策的 AI
- 一个追求“全自动成功率”的工具

这些都可能属于 **System 2.0+**，但**不属于现在**。

---

## 九、成功判定标准（System 1.0 Done）

System 1.0 只需要满足以下条件：

1. 我可以频繁使用它推进真实想法
2. 从想法到原型的路径明显变短
3. 我信任它的执行稳定性
4. 我愿意不断把新想法丢进去试

> **如果它只是一个“很好用的脚手架”，  
> 那就是 System 1.0 的成功，而不是失败。**

---

## 十、一句留给未来的注释

> **这个系统不是一开始就为所有人准备的，  
> 但如果它真的帮我变得更高效、更敢试，  
> 它自然会被别人需要。**
