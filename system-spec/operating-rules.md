# 系统运行规则（Operating Rules v1.1）

## 1) 何时必须打断 CEO
仅允许在以下情况向 CEO 提问：
- 业务方向变更
- 产品战略变更
- 成本支出新增/超预算

## 2) 何时无需确认（默认自动）
- 调研、方案探索、文档整理、代码修改、部署、测试、埋点、素材生成
- 不涉及方向/战略/成本的选型与细节决策：先探索后收敛，直接推进

## 3) 非强制确认问题的处理
前期可以询问；若 CEO 回复“始终确认”，则后续同类问题视为已确认并继续执行。

## 4) CEO 提问格式（强制）
【决策点】  
【现状】  
【分歧】A / B（各自收益与风险）  
【成本/风险】（如涉及）  
【你的选择】A / B / 停止 / 其他
