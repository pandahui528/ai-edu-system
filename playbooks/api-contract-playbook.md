# 接口契约 Playbook（减少联调成本）

## 统一响应结构（强制）
{
  "ok": true/false,
  "data": {...},
  "error": {"code": "...", "message": "..."} (当 ok=false),
  "traceId": "..."
}

## 必须约定
- 每个接口：请求字段、类型、必填、示例
- 错误码：最少 6 类（参数错误/鉴权/限流/超时/上游失败/未知）
- 上传接口：大小限制、超时、重试策略、返回可定位信息（traceId + code）
