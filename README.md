# ai-edu-system

这是一个一人公司 AI 原生教育创新项目“快速落地系统”，用于每月验证一个教育市场假设并交付可用原型。

## 如何使用
用户（CEO）只需用自然语言填写“项目初始化模板”，系统将按固定工作流推进，贯穿市场验证、MVP裁剪、工程落地、增长分发与复盘进化。

## 关键文件
- system-spec/global-system.md
- workflows/monthly-project-flow.md
- playbooks/*
- templates/*

## 最短使用步骤
1) 复制 templates/project-init.md 填写
2) 按 workflows/monthly-project-flow.md 运行
3) 项目结束填 templates/retrospective.md，并由全局复盘更新资产
